# AI-Powered Healthcare Analytics Dashboard

A full cloud-ready healthcare data analyst project built with Python, Streamlit, SQL logic, healthcare claims data, and AI-style executive summaries.

## Project Overview

This project analyzes healthcare claims, denials, prior authorizations, provider performance, operational KPIs, and high-value claim anomalies. It is designed for healthcare data analyst, clinical data analyst, and business intelligence analyst resumes.

## Features

- Claims analytics dashboard
- Denial reason trend analysis
- Provider performance reporting
- Prior authorization turnaround tracking
- High-value claim anomaly detection
- AI-generated operational summary
- Interactive filters by department, payer type, and claim status
- Cloud-ready Streamlit deployment

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- SQL concepts
- Healthcare claims analytics
- AI-assisted reporting
